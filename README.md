# Artificial Neural Networks & Deep Learning Lab Work

## Project Overview
This repository contains comprehensive lab work and projects for Artificial Neural Networks and Deep Learning (Semester 5). It features implementations of Perceptrons, Activation Functions, CNNs for MNIST/Image classification, GANs, GNNs, Autoencoders, and Self-Organizing Maps. Explore advanced topics like audio extraction, model training, and game environment design using TensorFlow and Keras.

## Structure
*   **MAM SADAF/**: Lab assignments covering Perceptrons, Activation Functions (Sigmoid, Tanh, ReLU), Convolutional Neural Networks (CNNs), Generative Models (GANs), Graph Networks (GNNs), Autoencoders, and Unsupervised Learning (SOM).
*   **ANN & DL/**: A structured project directory including code, data, documentation, models, and tests. Contains implementations for advanced models and game environment design.

## Technologies Used
*   Python
*   TensorFlow
*   Keras
*   MiniSom
*   Jupyter Notebooks
