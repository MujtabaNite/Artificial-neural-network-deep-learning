# Artificial-neural-network-deep-learning
Comprehensive lab work and projects for ANN &amp; Deep Learning (Semester 5). Features implementations of Perceptrons, Activation Functions, CNNs for MNIST/Image classification, GANs, GNNs, Autoencoders, and Self-Organizing Maps. Explore advanced topics like audio extraction, model training, and game environment design using TensorFlow and Keras.
